# Assets Directory

This directory should contain:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen (1284x2778px or similar)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)

For now, you can use placeholder images or generate them using:
- https://appicon.co/
- Figma or Sketch
- Or simply use solid color PNGs

**Temporary Solution:**
If you want to get the app running quickly, you can use Expo's default assets by temporarily removing the `icon`, `splash`, and `adaptiveIcon` references from `app.config.ts`, or create simple 1024x1024 PNG files with a solid color.
